Reactでfor文を用いるには

```javascript
push()
```
を用いる。
push()は配列の末尾に新しい要素を追加するためのメソッドです。
```javascript
fruits = ['apple', 'orange', 'banana'];
console.log(fruits);
// => ['apple', 'orange', 'banana']
fruits.push('strawberry');
console.log(fruits);
// => ['apple', 'orange', 'banana', 'strawberry']
```
このようにして用いる。
